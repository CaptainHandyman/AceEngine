# AceEngine
### Installation:
* ``` sudo apt-get install git ```
* ``` git clone https://github.com/RedBull4/AceEngine ```
* ``` cd AceEngine ```
* ``` sudo ./install.sh ```
### Uninstallation:
* ``` sudo make -f MakeFile uninstall ```
### Compilation:
```
g++ file.cpp -lAceEngine -lSDl2 \
                         -lSDL2_image \
                         -lSDL2_ttf -lGL 
```
